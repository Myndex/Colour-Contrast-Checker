# Colour Contrast Checker
Work in progress: Port over to APCA standards.


## CLI Commands

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:3000
yarn start

# build for production with minification
yarn build

# test the production build locally
yarn serve

# run tests with jest
yarn test
```
